## 🚀 About Project

**NoerAI** adalah website AI Assistant modern dengan tampilan **cyberpunk futuristik**.  
Project ini dibuat untuk membantu user dalam chatting dengan AI, membuat konten, merangkum materi, menyusun jadwal, dan menyimpan riwayat percakapan.

Project ini cocok untuk:

- Portfolio developer
- AI Assistant pribadi
- Web app eksperimen AI
- Project Next.js modern
- Demo aplikasi AI berbasis Groq

## 🧠 AI Modes

NoerAI memiliki beberapa mode yang dapat digunakan sesuai kebutuhan:

### 🤖 Chatbot Mode
Untuk percakapan umum, tanya jawab, coding, belajar, dan diskusi.

### ✨ AI Generator
Untuk membuat caption, artikel, ide konten, slogan, deskripsi produk, dan teks kreatif.

### 📚 AI Summary
Untuk merangkum materi panjang menjadi poin-poin penting yang mudah dipahami.

### 📅 AI Scheduler
Untuk membuat jadwal belajar, timeline tugas, rencana kerja, dan planning produktivitas.
